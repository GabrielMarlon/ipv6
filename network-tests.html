    Legenda
    • Score dual-stack Pontuação geral de 0 a 10. IPv4 ativo vale 3 pts, IPv6 ativo vale 5 pts e preferência por IPv6 vale mais 2 pts.
    • IPv4 Endereço público detectado via protocolo legado de 32 bits. Mostra IP, ISP e localização aproximada quando disponíveis, além da latência da requisição.
    • IPv6 Endereço público detectado via protocolo moderno de 128 bits. Sem IPv6, indica que o provedor não oferece o serviço ou que está bloqueado no caminho.
    • Protocolo preferido Quando IPv4 e IPv6 coexistem, qual o sistema operacional escolhe por padrão. IPv6 é o resultado ideal e indica que o Happy Eyeballs favorece o protocolo moderno.
    • DNS / AAAA Verifica se o resolver DNS consegue retornar registros AAAA (endereços IPv6) para um domínio. Usa DNS-over-HTTPS via 1.1.1.1 para evitar interferência de resolvers intermediários.
    • Performance / MTU Quando node server.js está rodando, executa PMTUD real: busca binária com ping ICMP e flag DF (Don't Fragment) para descobrir o MTU do caminho até 8.8.8.8. Sem servidor, mede throughput via HTTPS como aproximação.
